二叉树

  • 一个根节点
  • 每个节点最多只有2个子节点，分别为左节点、右节点

二叉搜索树

  • 左节点比父节点值小

实现类

  • 创建一个二分树类

class Node {
  constructor(key) {
    this.key = key
    this.left = null
    this.right = null
    this.root = null
  }
}

  • 往Bst添加值为8的节点

  • 继续Bst添加值为10的节点

  • 继续Bst添加值为6的节点

  • 最后结果应该： 8(6, 10)

  • 实现

class Bst {
  ...
  insert(key){
    let node = new Node(key)
    if (this.root) {
      // 新建一个函数，因为要递归处理
      this._insertNode(this.root, node)
    } else {
      this.root = node
    }
  }
  _insertNode(node, newNode) {
    if (newNode.key < node.key) {
      // 如果值没父节点的大， 靠左
      if (!node.left) {
        // 如果左节点还没
        node.left = newNode
      } else {
        // 如果左节点有了， 把左节点当作新的父节点，递归判断插入
        this._insertNode(node.left, newNode)
      }
    } else {
      // 如果值比父节点的大， 靠右
      if (!node.right) {
        // 如果右节点还没
        node.right = newNode
      } else {
        // 如果右节点有了， 把右节点当作新的父节点，递归判断插入
        this._insertNode(node.right, newNode)
      }
    }
  }
}

访问树大所有节点 - 遍历

中序、先序、后序

class Bst {
  ...
  // 中序
  inSort(cb) {
    const sort = (node, cb) => {
      if (node) {
        sort(node.left, cb)
        cb(node.key)
        sort(node.right, cb)
      }
    }
    sort(this.root, cb)
  }
  // 先序
  preSort(cb) {
    const sort = (node, cb) => {
      if (node) {
        cb(node.key)
        sort(node.left, cb)
        sort(node.right, cb)
      }
    }
    sort(this.root, cb)
  }
  // 后序
  postinSort(cb) {
    const sort = (node, cb) => {
      if (node) {
        sort(node.left, cb)
        sort(node.right, cb)
        cb(node.key)
      }
    }
    sort(this.root, cb)
  }
}

二叉搜索树中的值，搜索

找出最小值

左侧小，一直靠左一直小， 只要还有左侧的节点，就还有更小的值

class Bst {
  ...
  min() {
    if (this.root) {
      let node = this.root
      while (node && node.left !== null) {
        node = node.left
      }
      return node.key
    } else {
      return null
    }
  }
}

找出最大值

class Bst {
  ...
  max() {
    if (this.root) {
      let node = this.root
      while (node && node.right !== null) {
        node = node.right
      }
      return node.key
    } else {
      return null
    }
  }
}
