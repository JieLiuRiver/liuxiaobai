taro-ui

July 12, 2019

柳小白

样式

  • 入口
@import "./style/index.scss";
  • 小程序page节点默认flex样式
.page {
  display: flex;
  flex-direction: column;
  box-sizing: border-box;
  min-height: 100vh;
  background-color: #f8f8f8;
}
  • 全局view标签元素设置为怪异盒模型
view {
  box-sizing: border-box;
}
  • style目录下文件结构
    • index.scss 入口文件
    • variables变量管理
      • 它这里变量管理的方案是，$color-第一个词表明是颜色，$color-bg 、 $color-border第二个词表明是用在什么上的颜色， $color-bg-base第三个次表明这个颜色的定位，可根据自己的需求自定义
      • 这里主要定义的变量有以下方面的 
          $color-xxx 颜色
  $border-radius-xxx 边框倒圆
  $opacity-xxx 透明度
  $font-size 字体大小
  组件变量
  ....
    • themes 主题样式
    • mixins 混入(共用方法)
      • 里面有一个index.scss入口， libs目录分别管理不一样的mixin
      • 元素居中 
        /**
* 元素居中定位
*/
@mixin absolute-center($pos: absolute) {
  position: $pos;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
      • 超出省略 
        @mixin line($num: 1) {
  overflow: hidden;
  text-overflow: ellipsis;

  @if ($num == 1) {
    white-space: nowrap;
  } @else {
    display: -webkit-box;
    -webkit-line-clamp: $num;

    /* autoprefixer: off */
    -webkit-box-orient: vertical;
  }
}
      • 遮罩 
        @mixin overlay {
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  position: absolute;
  background-color: rgba($color: #000, $alpha: 0.3);
}
    • components 组件样式统一在这个目录下
      • 里面也有一个index.scss入口文件
